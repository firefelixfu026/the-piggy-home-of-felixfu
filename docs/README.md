## Archive

Historical development notes, older plans, temporary checklists, and troubleshooting records are stored in `docs/archive/`.

These files are kept for traceability, but they are not required for normal project review or deployment.

Current archive highlights:

- `docs/archive/project-history/`: old requirement notes, implementation analysis, next-step plans, and release checklists.
- `docs/archive/GITHUB_PUSH_TROUBLESHOOTING.md`: GitHub push troubleshooting notes.
- `docs/archive/DOCKER_DESKTOP_SETUP.md`: early local Docker setup notes.
